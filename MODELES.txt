========================================================================
  STRATÉGIE RECOMMANDÉE POUR T5.1
========================================================================
1. Importer d'abord ce qui mappe proprement en CC0 : les humanoïdes
   (gobelin, chevalier, éclaireur, chaman, nécromancien, porte-bouclier,
   orc), la gargouille (volante), le cyclope (géant), puis les
   tours/château.
2. Garder le FALLBACK PROCÉDURAL (déjà prévu par la tâche) pour :
   le bélier, le serpent, et tous les projectiles.
3. Animations : viser un jeu d'anims commun (marche/mort) et le
   retargeter sur les personnages plutôt que chercher chaque anim.
   La "Universal Animation Library" de Quaternius (CC0, 120+ anims dont
   locomotion et mort) est faite pour ça.
4. Cohérence visuelle : privilégier UN fournisseur dominant pour un
   style homogène (Quaternius = low-poly stylisé, cohérent avec le rendu
   actuel), et compléter au coup par coup ailleurs.

========================================================================
  SOURCES OPEN / CC0 (vérifiées le 04/07/2026)
========================================================================
Rappel licences :
  CC0        = domaine public, aucune attribution requise, usage
               commercial OK. Idéal.
  CC-BY      = libre MAIS attribution obligatoire (créditer l'auteur).
  CC-BY-SA   = attribution + partage à l'identique (contaminant).
  CC-BY-NC   = pas d'usage commercial → à éviter par principe.
Toujours vérifier la licence AU CAS PAR CAS sur la page du modèle.

1. QUATERNIUS — https://quaternius.com   [CC0]
   Le plus adapté : packs low-poly stylisés, personnages, MONSTRES
   ("Ultimate Monsters"), bâtiments, médiéval, nature. Formats OBJ/FBX/
   glTF + sources .blend. Tout en CC0.
   → Voir aussi la "Universal Animation Library" (CC0) pour les anims
     marche/mort retargetables.

2. KENNEY — https://kenney.nl   [CC0]
   40 000+ assets CC0. Excellents kits médiévaux/bâtiments (utile pour
   tours + château). Style un peu plus "jouet" que Quaternius.

3. POLY PIZZA — https://poly.pizza   [CC0 + CC-BY, filtrable]
   Agrégateur, sans compte, export glTF. Filtres "CC0" et "Animé".
   Recense déjà les monstres Quaternius pertinents ici : "Enemy Small",
   "Enemy Large", "Enemy Flying", "King", "Mech"...
   ATTENTION : contient aussi du CC-BY → vérifier la licence par modèle.
   Filtre direct : https://poly.pizza/explore?anim=1&lic=1

4. POLY HAVEN — https://polyhaven.com   [CC0]
   100% CC0. Surtout textures/HDRI (déjà utilisé dans le projet pour le
   ciel/les matières), quelques modèles.

5. SKETCHFAB (filtre CC0) — https://sketchfab.com   [CC0 + CC-BY, mixte]
   Beaucoup de choix, mais licences très variables. NOTE : la boutique
   payante a fermé (la vente est passée sur Fab d'Epic), MAIS les
   téléchargements Creative Commons/CC0 gratuits fonctionnent toujours.
   Conseil : télécharger ET ARCHIVER ce qu'on prend (ne pas dépendre
   d'un lien qui pourrait disparaître).

6. Méta-liste CC0 (pour aller plus loin) —
   https://github.com/madjin/awesome-cc0
   Liste maintenue de ressources CC0 (modèles, textures, audio).

------------------------------------------------------------------------
  NOTE HONNÊTE
------------------------------------------------------------------------
- La liste des modèles est SÛRE (déduite des entités réelles du jeu).
- Les licences ci-dessus ont été vérifiées le 04/07/2026, mais elles
  peuvent changer et varient par modèle : re-vérifier sur chaque page.
- Les créatures "génériques" (gobelin, orc, géant, volant) se trouvent
  bien en CC0 ; les créatures TRÈS spécifiques (cyclope exact, gargouille
  exacte, nécromancien nommé) sont plus rares — on prend souvent un
  modèle proche qu'on adapte (couleur/échelle), ce que le jeu fait déjà.
